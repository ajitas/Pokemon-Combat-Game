Goal: To defeat all the pokemons.

How to play: Select a pokemon from the four pokemons shown to play with. The rest will be considered as enemies.

Select one enemy from the enemy list and start the battle. If you lose, the game restarts. If you win, you can select more from the enemy lot to play with, until you defeat all of them

If you defeat all the enemies, you win. you can play the game again if you wish.
